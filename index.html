How Both Game Modes Work

  1. A room supports 2–6 players, each on their own device. The host chooses the starting hand size.
  2. Typing is used only during Build Together, when everyone can add Problem and Solution Cards to the deck.
  3. 3 Fold Cord: A rotating Round Leader draws one shared Problem Card. Every player—including the leader—privately plays exactly one Solution Card from their hand.
  4. After everyone locks in, all played cards are revealed with player names. Players explain aloud how their card could help and what principle supports it.
  5. Everyone votes Agree, Disagree, or Needs More Discussion. All Agree earns one team point. Any Disagree ends the round with no point. Needs More Discussion keeps the same cards and Problem open for another discussion and vote.
  6. Played cards are discarded after an Agree or Disagree result, and every player draws back up to the host-selected hand size. The team wins when it reaches 12 points.
  7. Problem? Solved!: One Problem Picker draws. Everyone else secretly plays one Solution Card. The Picker chooses the best anonymous card; that player earns one point. First to five points wins.

Saved games store the custom card deck. The host can use the same deck for 3 Fold Cord or Problem? Solved! each time.
