How Both Game Modes Work

  1. A room supports 2–6 players, each on their own device.
  2. In Build Together, everyone gets the same timer and can add unlimited Problem and Solution Cards.
  3. 3 Fold Cord: A rotating Round Leader draws the Problem. Every player submits a proposed solution and the principle behind it. A Solution Card may be used as inspiration.
  4. When everyone has answered, all ideas are revealed with player names. Discuss them together, then everyone votes.
  5. The team earns one point only when every connected player approves. Reach 12 points before three rounds need more work.
  6. Problem? Solved!: One Problem Picker draws. Everyone else secretly plays a Solution Card and may explain it.
  7. The Picker chooses the best anonymous answer. That player earns one point, and the Picker rotates. First to five points wins.

Saved games store the custom card deck. The host can use the same deck for 3 Fold Cord or Problem? Solved! each time.
